get_name(agent)

Process the robots (when available) found on the user agent strings and returns its name: 

  user_agent = "Googlebot/2.X (+http://www.googlebot.com/bot.html)"
  Sitealizer::Parser::Robot.get_name(user_agent)
  => "Googlebot"
