RUWA is a simple web analytics in ruby on rails. It‘s not complete and there may be bugs. The code is inspired by Piwik.org

To install, launch the ./script/server and visit localhost:/users/add_user

The first user will be the administrator

Once logged in, the administrator can create new users.

Create a new user and then logout.

Go to /login/login and login using the administrator. There you can create a new User. Once the user is created, you want to logout again and login with User. After inserting the project name, you can create the project.

This will take a while, since every time a new User is created, 500 random generated visitors are created for testing purpose.

Every user has a project associated with him. The project_id will be used by the javascript tracker code.

To install a tracker in a website, you have to add this code (taken from Piwik.org, a excellent open source web analytics in PHP)

<script type="text/javascript"> var pkBaseURL = 

        (("https:" == document.location.protocol) ? "https://thedomain-with-ruwa:4000/javascripts/" : "http://thedomain-with-ruwa:4000/javascripts/");

document.write(unescape("%3Cscript src=’" + pkBaseURL + "piwik.js’ type=‘text/javascript’%3E%3C/script%3E")); </script>

<script type="text/javascript"> 

        piwik_action_name = document.location.href;
        piwik_idsite = 1;
        piwik_url = pkBaseURL + "log.js";
        piwik_log(piwik_action_name, piwik_idsite, piwik_url);

</script>

The piwik_idsite variable holds the project_id associated with the User.
